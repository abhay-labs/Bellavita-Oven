# BellaVita Oven

A full-stack bakery and dessert ordering web application focused on clean UI, structured backend, and seamless ordering workflow.

---

## Overview

BellaVita Oven enables users to browse bakery products, manage cart items, and place orders efficiently.  
The project follows a modular architecture with clear separation between frontend and backend components.

---

## Features

- Product listing (cakes, pastries, desserts)
- Cart management
- Checkout flow
- REST API integration
- Responsive design
- Offer and pricing display

---
